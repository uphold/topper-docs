# User

| Event | Trigger |
| - | - |
| [`user:created`](#usercreated) | User created a new account. |
| [`user:logged-in`](#userlogged-in) | User logged in. |
| [`user:kyc:identity:initiated`](#userkycinitiated) | User's KYC identity process has been initiated. |
| [`user:kyc:identity:succeeded`](#userkycsucceeded) | User's KYC identity process has succeeded. |
| [`user:kyc:identity:failed`](#userkycfailed) | User's KYC identity process has failed. |

## `user:created`

Triggered when a new user is created.

<Tabs>
  <TabItem label="Schema" value="schema" default>

- `user`: User object.
  - `id`: UUID of the user.
  - `createdAt`: Timestamp at which the user was created.

</TabItem>
  <TabItem label="Example" value="example">

```js
{
  "name": "user:created",
  "id": "00a991cf-a870-43af-88b8-43c51532831d",
  "bootstrapTokenId": "141bfa06-481e-4684-96eb-cec4ad529616",
  "data": {
    // highlight-start
    "user": {
      "id": "b9fe022b-f436-49e1-bb89-6f2e8eabf336",
      "createdAt": "2023-06-12T17:21:21.240Z"
    }
    // highlight-end
  }
}
```

  </TabItem>
</Tabs>

## `user:logged-in`

Triggered when a user logs in.

<Tabs>
  <TabItem label="Schema" value="schema" default>

- `authenticationMethod`: Authentication method used (`otp` or `sso`).
- `user`: User object.
  - `id`: UUID of the user.
  - `confirmedAt`: Timestamp at which the user was confirmed.

</TabItem>
  <TabItem label="Example" value="example">

```js
{
  "name": "user:logged-in",
  "id": "00a991cf-a870-43af-88b8-43c51532831d",
  "bootstrapTokenId": "141bfa06-481e-4684-96eb-cec4ad529616",
  "data": {
    // highlight-start
    "authenticationMethod": "otp",
    "user": {
      "id": "b9fe022b-f436-49e1-bb89-6f2e8eabf336",
      "confirmedAt": "2023-06-12T17:21:21.240Z"
    }
    // highlight-end
  }
}
```

  </TabItem>
</Tabs>

## `user:kyc:identity:initiated`

Triggered when a user's KYC process has been initiated.

<Tabs>
  <TabItem label="Schema" value="schema" default>

- `kycSession`: KYC session object.
  - `initiatedAt`: Timestamp at which the KYC session was initiated.
- `user`: User object.
  - `id`: UUID of the user.

</TabItem>
  <TabItem label="Example" value="example">

```js
{
  "name": "user:kyc:identity:initiated",
  "id": "00a991cf-a870-43af-88b8-43c51532831d",
  "bootstrapTokenId": "141bfa06-481e-4684-96eb-cec4ad529616",
  "data": {
    // highlight-start
    "kycSession": {
      "initiatedAt": "2024-01-15T10:30:00.000Z"
    },
    "user": {
      "id": "b9fe022b-f436-49e1-bb89-6f2e8eabf336"
    }
    // highlight-end
  }
}
```

  </TabItem>
</Tabs>
